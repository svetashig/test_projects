# test_projects
FigureViewer тестовое задание по заданному ТЗ

ТЗ:
Необходимо привести пример программы или библиотеки на C++,
которая выводит на экран различные геометрические фигуры: круг, квадрат, треугольник, прямоугольник и т.п.
Глубина проработки примера (количество фигур, параметры, методы) на Ваше усмотрение.
Программа не обязательно должна запускаться и работать (хотя это будет плюсом). 


Описание:
В меню Рисование можно выбрать текущую Фигуру.

По клику в окне создается фигура, выбранной формы:
    - для круглых фигур Курсор в центре фигуры;
    - для прямоугольных фигур Курсор в верхнем левом углу;
    
Цвет задается рандомно.
Размеры фигур также рандомно задаются.
Для Точки размер фиксированный 3 px, т.к. фактически это маленький круг.

Фигуры -> Очистить экран для очистки области рисования.

P.S.
Программа собирается и запускается, также положила собранную realese-версию.
